### How to use this repository

1. Clone the repository
2. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```
3. Run Jupyter Notebook:
   ```bash
   jupyter notebook
   ```
4. Open the notebook file `fathomnet-download-images.ipynb` in Jupyter Notebook.
5. Follow the instructions in the notebook to run the code.
6. The images will be downloaded to the `images` directory in the current working directory.
7. Open the `fathomnet-hierarchical_test-split.ipynb` in Jupyter Notebook.
8. Follow the instructions in the notebook to run the code.
9. Model will be saved in the `outputs` directory in the current working directory.
