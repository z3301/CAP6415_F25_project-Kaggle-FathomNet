### How to use this repository

1. Clone the repository
2. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```
3. Open the notebook file `fathomnet-download-images.ipynb` in Jupyter Notebook or any compatible environment.
4. Follow the instructions in the notebook to run the code.
5. The images will be downloaded to the `images` directory in the current working directory.
6. Open the `fathomnet-hierarchical_test-split.ipynb` in Jupyter Notebook or any compatible environment.
7. Follow the instructions in the notebook to run the code.
8. Model will be saved in the `outputs` directory in the current working directory.
